# Markdown Quick Search

## Basic Syntex

| Item | Syntex |
| :-: | :-: |
| Heading | `# Head` |
| Bold | `**Bold Text**` |
| Italic | `*Italic Text*` |
| Blockquote | `> blockquote` |
| Ordered List | `1.Item` |
| Unordered List | `- Item` |
| Code Block | ` ```Language``` ` |
| Horizonal Line | `---` |
| Link | `[title](https://your.link)` |
| Image | `![Alt Text](Image Path)` |
